# Занятие: CVZone — компьютерное зрение на Python

Материал разбит на отдельные файлы по темам, чтобы студенты могли
запускать и разбирать каждый пример отдельно, не листая один большой файл.

## Установка

```bash
pip install cvzone
pip install mediapipe==0.10.9
pip install djitellopy   # нужен только для файла 08 (дрон Tello)
```

> Важно: версии cvzone и mediapipe должны быть совместимы друг с другом.
> Если при импорте cvzone падает ошибка, связанная с mediapipe —
> переустановите mediapipe именно на версию 0.10.9.

## Структура файлов

| Файл | Тема |
|---|---|
| `01_basic_drawing.py` | Рамки со скруглёнными углами и текст с подложкой (cornerRect, putTextRect) |
| `02_image_tools.py` | Загрузка изображений по URL, наложение PNG с прозрачностью, поворот и склейка кадров |
| `03_color_and_contours.py` | Поиск объектов по цвету (HSV) и поиск контуров |
| `04_face_detection.py` | Детекция лиц через веб-камеру |
| `05_face_mesh_and_segmentation.py` | Сетка лица (468 точек) и замена фона (segmentation) |
| `06_hand_and_pose.py` | Трекинг рук (жесты, счёт пальцев) и трекинг позы тела |
| `07_plot_module.py` | Живой график в реальном времени (синус и координата лица) |
| `08_tello_face_detection.py` | Lab 31: детекция лиц на видео с дрона Tello |

## Как использовать на занятии

1. Начните с `01` и `02` — это база (рисование, работа с изображениями),
   камера не обязательна.
2. `03`–`06` — требуют веб-камеру, показывают детекцию по цвету, лицам,
   рукам и позе.
3. `07` — можно показать отдельно как визуализацию данных.
4. `08` — финальный пример, объединяющий детекцию лиц с реальным дроном.
   Требует подключения к Wi-Fi дрона Tello перед запуском.

## Важное замечание про индекс камеры

В исходных примерах использовался `cv2.VideoCapture(2)` — это означает
"третья камера по счёту" и подходит только для конкретного компьютера
автора. Для большинства ноутбуков со встроенной камерой правильный
индекс — `0`. Во всех файлах ниже стоит `0` по умолчанию, с комментарием,
как поменять.
