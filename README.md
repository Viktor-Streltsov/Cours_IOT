# Курс IoT: Программирование дрона DJI Tello на Python

Практический курс по управлению дроном **DJI Tello** с помощью Python и библиотеки `djitellopy`. Курс охватывает основы языка, принципы управления беспилотником и построение модульных проектов.

## Подробнее по урокам

### Lesson 1 — Основы Python
Базовый синтаксис: переменные, строки, списки, словари, функции, модули. Упражнения: калькулятор, работа со списками, поиск слов.

### Lesson 2 — Первый полёт
Подключение к дрону по Wi-Fi, проверка заряда батареи, взлёт (`takeoff`), ожидание (`sleep`), посадка (`land`).

### Lesson 3 — Движение вверх/вниз
Команды `move_up` и `move_down`. Защита от вылета при низком заряде батареи.

### Lesson 4 — Движение влево/вправо
Команды `move_left` и `move_right`.

### Lesson 5 — Движение вперёд/назад + функции
Команды `move_forward` и `move_back`. Вынесение проверки батареи в отдельную функцию `check_battery()`.

### Lesson 6 — Повороты
Команды `rotate_clockwise` и `rotate_counter_clockwise`. Полёт по маршруту с разворотами (U-turn).

### Lesson 7 — Циклы и try/except
Цикл `for` для многократных манёвров (прыжки на разные высоты). Обёртка полёта в блок `try/except` для безопасной аварийной посадки.

### Lesson 8 — Вложенные циклы и полёт по фигурам
Универсальная функция `fly_shape(sides, steps, step_distance, angle)` для полёта по квадрату, треугольнику и другим многоугольникам. Полный цикл `try/except/finally`.

### Lesson 9 — Акробатика (Flip)
Выполнение флипа (`flip_right`) с обработкой ошибки на случай недостаточного заряда или высоты.

### Lesson 10 — Консольное меню + модульная структура
Интерактивное меню управления дроном из консоли. Проект разбит на пакеты:
```
dron_lesson_10/
├── main.py              # Точка входа, главное меню
├── drone/
│   ├── connect.py       # Подключение к дрону
│   ├── movement.py      # Функции движения
│   └── safety.py        # Проверка батареи
├── missions/
│   ├── square.py        # Миссия: полёт по квадрату
│   └── patrol.py        # Миссия: патрулирование
└── utils/
    └── logger.py        # Логирование
```

### Lesson 11 — RC-управление (в разработке)
Основа для управления дроном в реальном времени через `send_rc_control`.

### Lesson 12 — 3D-траектория
Команда `go_xyz_speed(x, y, z, speed)` для движения по координатам в пространстве. Полёт по диагоналям и сложным 3D-маршрутам.

## Требования

- Python 3.9+
- Дрон DJI Tello (подключение по Wi-Fi)
- Библиотека `djitellopy`

### Установка зависимостей

```bash
pip install djitellopy
```

---

## Быстрый старт

1. Включите дрон DJI Tello
2. Подключитесь к Wi-Fi сети дрона (`TELLO-XXXXXX`)
3. Запустите нужный скрипт:

```bash
python Lesson_2/lesson_2.py
```

> **Важно:** Перед каждым полётом убедитесь, что заряд батареи выше 20%. Все скрипты проверяют уровень заряда автоматически.

## Используемые команды Tello

| Команда | Описание |
|---------|----------|
| `tello.connect()` | Подключение к дрону |
| `tello.get_battery()` | Уровень заряда батареи (%) |
| `tello.takeoff()` | Взлёт |
| `tello.land()` | Посадка |
| `tello.move_up(cm)` | Подъём на N сантиметров |
| `tello.move_down(cm)` | Снижение на N сантиметров |
| `tello.move_forward(cm)` | Движение вперёд |
| `tello.move_back(cm)` | Движение назад |
| `tello.move_left(cm)` | Движение влево |
| `tello.move_right(cm)` | Движение вправо |
| `tello.rotate_clockwise(deg)` | Поворот по часовой стрелке |
| `tello.rotate_counter_clockwise(deg)` | Поворот против часовой стрелки |
| `tello.flip_right()` | Флип вправо |
| `tello.go_xyz_speed(x, y, z, speed)` | Движение по координатам 3D |
